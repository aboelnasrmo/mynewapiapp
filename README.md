# apiapp

A new Flutter API project.

## Getting Started

Application to call API by using Flutter_bloc , and the following packages :
  - build_runner
  - dio
  - get_it
  - json_annotation
  - json_serializable
  - retrofit
  - retrofit_generator
  
  
  
  The API used for the call is from the following link :
  https://gorest.co.in/public/v2/
  
  the endpoint is : users
